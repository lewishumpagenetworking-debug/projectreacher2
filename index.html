Cloud Account

Data Health / Backup

Export weekly. Uploading a file here imports its historical data (workouts, meals, logs) into your current app — it never changes your current program, features, or app version.

Advanced: Full Restore

Replaces your entire current app state with the uploaded file, exactly as it was exported — this can remove newer features (like Day 6) if the file predates them. Only use this if you specifically want to roll back to an old snapshot. A safety backup of your current data is still taken first.

App Updates

If a feature you know was added (like Day 6) isn't showing up, your browser may be holding onto an old cached copy of the app. This clears that cache and reloads the latest version — your saved data is not touched or deleted, only the app's own code files.

AI Specialists

Claude-powered coaching layer. Read-only by default: it can analyse, explain and propose changes, but never edits your program, nutrition, supplements or appearance routine automatically — every write requires your explicit accept in Proposed Changes.

Claude Integration

Aesthetic Protocol is a static site with no backend server. This connects directly from your browser to Anthropic's API using your own API key, stored only in this browser's local storage. Anyone with access to this device/browser could read that key. This is a deliberate trade-off you're choosing, not the server-side setup Anthropic recommends for production apps.
